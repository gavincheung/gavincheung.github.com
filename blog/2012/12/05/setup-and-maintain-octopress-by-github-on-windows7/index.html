在Windows7中用Github搭建和维护Octopress

安装必要软件

  1. git - 用于在Github上维护Octopress

    • 最新版本的git安装程序可以在 msysgit 找到；
    • 本文发布时的最新版本为Git-1.8.0-preview20121022，下载并安装之。

  2. ruby - Octopress的运行环境

    • ruby安装程序可以在 RailsInstaller 找到；
    • Octopress要求安装 rubyinstaller-1.9.2-p290，下载并安装之；
    • 确保C:\Ruby192\bin是否在Windows当前用户的PATH变量环境中。

  3. 安装Devkit - 编译环境

    • 下载 RubyInstaller DevKit，并按如下方法安装：
      • 将 DevKit 自解压包释放到 C:\DevKit；
      • 在 Windows CMD 窗口中 cd 到Devkit的解压目录并执行：ruby dk.rb init；
      • 执行：ruby dk.rb install。

  4. 安装python - 代码加亮系统pygments需要python支持

    • 最新版本的ActivePython安装程序可以在 ActivePython 找到；
    • 本文发布时的最新版本为ActivePython 2.7.2，下载并安装之;
    • 安装完毕后，在 Windows CMD 窗口中 cd 到ActivePython的Script目录并执行：easy_install pygments；
